Code Examples

Create a Project (JavaScript)
const { InsporaClient } = require('@inspora/sdk');

const client = new InsporaClient({
  apiKey: 'your-api-key',
  environment: 'production'
});

const project = await client.projects.create({
  name: 'My New Project',
  description: 'Project created via API',
  teamId: 'team-123'
});

console.log('Project created:', project.id);
Create a Project (Python)
from inspora import InsporaClient

client = InsporaClient(
    api_key='your-api-key',
    environment='production'
)

project = client.projects.create(
    name='My New Project',
    description='Project created via API',
    team_id='team-123'
)

print(f'Project created: {project.id}')
